End-to-end invoice automation using Google Forms, Sheets, Apps Script, Docs, Drive, and Gmail.

## Features
- Auto PDF invoice generation
- Room + Food invoices
- GST calculation
- Google Drive folder structure (Year/Month)
- Year and Month folder structure will be created automatically in the Main Invoices folder.
- Automated email delivery

## Tech Stack
- Google Forms
- Google Sheets
- Google Apps Script
- Google Docs
- Gmail API

## Repository Purpose
This repository contains:
- Apps Script
- Google Doc template placeholders
- Setup & form documentation

No external dependencies required.
